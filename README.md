# 🍕 Muku Pizza Hut & Cafe — Official Website

> **"The Best of Taste"** — Jaipur's favourite 100% vegetarian fast food cafe.

---

## 📌 About the Project

This is the official single-page website for **Muku Pizza Hut & Cafe**, located in Jhotwara, Jaipur, Rajasthan. The site is built as a single HTML file with embedded CSS and JavaScript — no frameworks or build tools required.

---

## ✨ Features

- **Animated Intro Screen** — 3D logo spin with particle background (Canvas API)
- **3D Hero Section** — Interactive floating orbs using Three.js, react to mouse/touch movement
- **Tabbed Menu** — Categories: Pizza, Burger, Sandwich, Beverages, Garlic Bread, Maggi, and more
- **Chef's Specials Section** — Highlighted signature items
- **About Us Section** — Cafe story and highlights
- **Contact Section** — Address, phone, and WhatsApp order button
- **Sticky Navigation** — Smooth scroll with blur backdrop
- **Floating WhatsApp Button** — Always-visible "ORDER NOW" CTA
- **Fully Responsive** — Mobile-friendly layout with fluid typography
- **Scroll Reveal Animations** — Sections animate in as user scrolls

---

## 🛠️ Tech Stack

| Technology | Usage |
|---|---|
| HTML5 | Structure and markup |
| CSS3 | Styling, animations, layout (Grid & Flexbox) |
| Vanilla JavaScript | Interactions, tab switching, scroll animations |
| [Three.js r128](https://threejs.org/) | 3D hero background (WebGL) |
| Canvas API | Particle animation on intro screen |
| Google Fonts | Playfair Display, Bebas Neue, Nunito |

## 🚀 How to Run

No installation needed. Just open the file in any modern browser:

```bash
# Option 1 — Double-click the file
Muku_cafe_website7.html

# Option 2 — Use VS Code Live Server extension
Right-click → "Open with Live Server"

# Option 3 — Python local server
python -m http.server 8000
# Then visit http://localhost:8000/Muku_cafe_website7.html
```
